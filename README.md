# PythonQtBrowser

![image](https://user-images.githubusercontent.com/34125174/208937208-571d4934-cb2b-455d-879b-f8422ef2aa62.png)

This is a browser made in python with Qt using a PyQt5

run `main.py` to start

##### Available Features:
- multi tab support
- it works

##### Planned Features:
- Zoom support

- Fullscreen support


#### Notes:
- As of 21/11/2020, you cannot login into google
- It is kind of buggy, so please report the bugs

- It is slow

## Dependencies:
- Python
- PyQt5
- colorama
- PyQtWebEngine
- adblockparser

Enter `pip install -r requirements.txt` to install the dependencies
